Limpieza quincenal — proceso manual en Supabase

Por seguridad, el borrado de registros no se puede hacer desde este panel. Esto protege la evidencia: nadie puede eliminar datos desde la app pública, ni siquiera con la contraseña de admin de esta página. Solo se puede borrar entrando directamente a tu cuenta de Supabase.

  1. 1. Exporta primero con el botón de arriba y confirma que el archivo se descargó bien
  2. 2. Entra a supabase.com → tu proyecto JX_HYDROLIT
  3. 3. Ve a Table Editor → abre checkins o gps_logs
  4. 4. Filtra por fecha si quieres revisar el rango exacto de la quincena
  5. 5. Selecciona las filas con el checkbox de la izquierda
  6. 6. Clic en Delete rows (botón rojo) y confirma
